# bitGAN Diffusion
bitGAN Diffusion
bitGAN Diffusion v1.0 by Mindshift and Right on Ricky
is a custom-trained unconditional diffusion model trained on original dataset of 8,000+ bitGAN images using the fine-tuning openai diffusion model notebook by Alex Spirin and based on the Pixel Art Diffusion 3.0 framework by KaliYuga

bitGAN Diffusion V.1 runs within a fork of Disco Diffusion 5.4.
